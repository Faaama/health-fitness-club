Requirements
==============
- Node.js
- PostgreSQL

Steps to Set Up Project
=========================
1. Clone the Git Repository
2. Set up the Database
	- Using pg4 Admin, create a new database FitnessClubSystem
	- Open the DDL.sql, DML.sql and View.sql files and execute them (in order)
3. Set up the Application
	- Open the terminal and run "npm install" and "npm i pg"
4. Run the command "node ./server.js" to start the server and visit https://localhost:3000/login
